# External Skills Reference

このディレクトリは外部Agent-skillsの参考情報置き場である。

## ルール

- 外部Skillを丸ごとコピーしない。
- 出典不明のルールを採用しない。
- `AGENTS.md`、`skills/00-core`、プロジェクト固有Skillと矛盾する内容は採用しない。
- 採用する場合は、内容を確認し、このプロジェクト向けに再編集してから `skills/` の該当カテゴリへ取り込む。
- secret、token、ライセンス不明素材、不要PIIを置かない。

## 使い方

1. 外部Skillの出典、ライセンス、目的を確認する。
2. このプロジェクトの `AGENTS.md` と競合しないか確認する。
3. 必要な部分だけ要約し、自分用Skillとして再編集する。
4. 採用理由とレビュー結果を `docs/review/` へ保存する。
