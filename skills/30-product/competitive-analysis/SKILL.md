# Competitive Analysis

## Purpose

AI PM Platformの勝ち筋を、競合と市場の観点から評価する。

## When to use

- 市場調査、競合分析、勝ち筋整理。
- MVPや価格戦略を見直すとき。
- 投資家/顧客向け資料の前段。

## Inputs

- 対象市場。
- 競合リスト。
- ユーザー課題。
- 自社の機能と制約。

## Process

1. 市場カテゴリを定義する。
2. 競合を直接/隣接/代替に分ける。
3. SWOTまたはFive Forcesで整理する。
4. AI PMとしての差別化を明確にする。
5. MVPと価格仮説へ接続する。

## Output

- 競合比較。
- 勝ち筋。
- リスク。
- 次の検証Issue。

## Constraints

- 必要に応じて最新情報を確認する。
- 推測と事実を分ける。
- 競合を過小評価しない。

## Checklist

- [ ] 競合分類がある。
- [ ] 差別化が明確。
- [ ] 市場リスクがある。
- [ ] 次の検証がある。
